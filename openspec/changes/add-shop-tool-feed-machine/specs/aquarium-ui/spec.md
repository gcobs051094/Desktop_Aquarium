## ADDED Requirements

### Requirement: 商店「工具」Tab
商店覆蓋視窗（ShopOverlay）SHALL 提供「工具」tab，用於顯示可解鎖之工具項目，並與既有 tab 一致以卡片列表呈現（含預覽圖、名稱、說明、狀態/條件、操作控制）。

#### Scenario: 工具 tab 顯示
- **WHEN** 使用者開啟商店覆蓋視窗
- **THEN** 商店提供「工具」tab
- **AND** 工具 tab 內容為可捲動的卡片列表

### Requirement: 工具「飼料投食機」之解鎖前後 UI
系統 SHALL 在商店「工具」tab 顯示「飼料投食機」項目；未解鎖時顯示解鎖條件並禁用顏色操作；解鎖後以「顏色下拉式選單」提供顏色調整，且預設顏色為灰黑色。

#### Scenario: 未解鎖時顯示條件
- **WHEN** 使用者開啟商店且投食機尚未解鎖
- **THEN** 投食機卡片顯示預設灰黑色預覽
- **AND** 顯示解鎖條件「獲得 10 隻成年孔雀魚」（以里程碑判定）
- **AND** 顏色調整控制為不可操作狀態

#### Scenario: 解鎖後顯示顏色下拉
- **WHEN** 投食機已解鎖且使用者開啟商店
- **THEN** 投食機卡片顯示顏色下拉式選單
- **AND** 下拉選單預設選擇「灰黑色」

#### Scenario: 切換顏色即時生效
- **WHEN** 投食機已解鎖且使用者在下拉選單選擇新的顏色
- **THEN** 水族箱中的投食機顯示 SHALL 立即更新為對應顏色素材

### Requirement: 水族箱顯示投食機素材
投食機解鎖後，系統 SHALL 在水族箱座標 (25, 240) 顯示投食機素材圖片，並隨顏色選擇更新顯示。

#### Scenario: 解鎖後顯示投食機
- **WHEN** 投食機解鎖條件已達成
- **THEN** 水族箱在座標 (25, 240) 顯示投食機圖片
