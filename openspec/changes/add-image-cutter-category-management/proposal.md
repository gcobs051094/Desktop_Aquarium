# Change: Add Image Cutter Category Management

## Why
使用者希望在 DFS 偵測後，能以更彈性的方式自訂類別與分配區域：透過新增類別、在來源影像上以滑鼠拖移框選 DFS 區域來分配類別，並將裁切結果輸出到來源影像的同級目錄下，以類別名稱建立資料夾，類別內則為 DFS 偵測到的多張影像。

## What Changes
- **類別管理 UI**：DFS 偵測後顯示類別清單、類別名稱輸入框（作為資料夾名稱）、新增類別按鈕、刪除類別按鈕
- **互動式區域分配**：點擊類別清單中的類別後，在來源影像上顯示十字線游標，按住滑鼠拖移可繪製 bbox，釋放後將框選到的 DFS 區域分配到該類別
- **視覺化**：不同類別以不同顏色標記，未分配區域以灰色顯示；類別清單顯示各類別名稱與區域數量
- **輸出目錄**：自訂類別模式下，輸出到來源影像的同級目錄，在該目錄下依類別名稱建立子資料夾，類別內依左到右順序命名裁切檔（`base_name_000.png` 等）

## Impact
- **修改能力**：圖片處理工具（image-processing）
- **受影響的代碼**：
  - 修改 `tools/image_cutter_gui.py` - 新增類別管理 UI、預覽滑鼠交互（十字線、bbox 拖移）、座標轉換、自訂類別裁切與輸出邏輯
  - 保留 `tools/alpha_dfs_crop.py` - DFS 算法不變
- **受影響的目錄**：`tools/` 目錄
- **非 BREAKING**：原有「不分類」「魚」等輸出模式仍可使用；自訂類別為新增選項
