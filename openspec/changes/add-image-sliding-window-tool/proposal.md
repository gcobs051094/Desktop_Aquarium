# Change: Add Image Sliding Window Tool

## Why
在資料處理階段，需要將 `resource/` 目錄中的雪碧圖（sprite sheet）按照固定的滑窗尺寸進行裁切，以便後續進行人工分類。目前資源目錄中包含多個精靈圖的雪碧圖，需要一個工具來自動化裁切過程，提高資料處理效率。

## What Changes
- 新增一個命令行工具，用於滑窗圖片裁切
- 支援指定滑窗的寬度和高度
- 支援指定滑窗的步長（stride）
- 自動遍歷 `resource/` 目錄中的所有圖片文件
- 將裁切後的圖片保存到指定的輸出目錄
- 支援自訂輸出文件命名規則

## Impact
- **新增能力**：圖片處理工具（image-processing）
- **受影響的代碼**：新增工具腳本文件
- **受影響的目錄**：`tools/` 或專案根目錄（工具腳本位置）
