# Change: 新增寵物系統與龍蝦寵物

## Why
使用者需要能夠在水族箱中召喚寵物，寵物具有獨特的功能來增強遊戲體驗。龍蝦寵物作為第一個寵物，能夠在水族箱底部走動並自動拾取落下的金錢，為玩家提供便利。寵物需要通過商店解鎖，且每種寵物只能召喚一隻。

## What Changes
- **寵物系統基礎**：新增寵物類別系統，支援寵物在水族箱中的顯示、移動和行為邏輯。
- **商店寵物商品**：在商店中顯示寵物商品，支援解鎖狀態顯示和購買功能。
- **寵物解鎖機制**：實作寵物解鎖條件檢查（龍蝦需要5隻large_guppy解鎖）。
- **龍蝦寵物**：實作龍蝦寵物類別，在水族箱底部水平走動，自動拾取觸碰到的金錢。龍蝦顯示大小設定為 scale=0.6（較小尺寸）。
- **寵物召喚限制**：確保每種寵物只能召喚一隻。

## Impact
- **受影響能力**：水族箱使用者介面（aquarium-ui）、遊戲經濟（game-economy）、寵物行為（pet-behavior）
- **受影響代碼**：
  - `aquarium_window.py`：新增寵物列表管理、寵物繪製、寵物更新邏輯；商店中新增寵物商品顯示與購買邏輯
  - `config.py`：新增寵物配置（解鎖條件、召喚條件等）
  - 新增 `pet.py`：寵物基礎類別與龍蝦寵物實作
